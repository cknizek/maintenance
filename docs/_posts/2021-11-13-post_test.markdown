### Model number encoding
~~~
ABCCCDDDD
1. <A> is the current wing. A valid value is *W*, *C*, or *E*.
2. <B> is the current floor.  A valid value is *0*, *1*, *2*, or *3*.
3. <CCC> is the type of electrical component. A reference table has been included below.
4. <DDD> is the serial number of the electrical component. A valid value is any unique integer in the range 0-9999. 
>>> Example: W1SPL0001 = <W><1><SPL><0001>


~~~
